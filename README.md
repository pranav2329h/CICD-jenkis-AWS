# Launchpad

A simple responsive Next.js application built with the App Router and TypeScript.

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Production

```bash
npm run lint
npm run build
npm start
```

## Docker

```bash
docker build -t nextjs-launchpad .
docker run -p 3000:3000 nextjs-launchpad
```
